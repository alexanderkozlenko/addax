Remarks

An instance of the generated record handler will be automatically added to the Handlers shared collection, which is used to resolve the required handler for reading and writing records. Use the TabularFieldOrderAttribute attribute to map a type member to a tabular field, and optional attributes TabularFieldNameAttribute and TabularConverterAttribute to define a field name and apply a value converter.
